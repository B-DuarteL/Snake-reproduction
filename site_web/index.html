Documentation 'Snake'

Voici la documentation du projet snake

Présentation du projet

Le projet "Snake" est un jeu programmé en Python avec la bibliothèque Tkinter. Le joueur contrôle un serpent qui se déplace sur une grille et doit manger des pommes pour augmenter son score.

Origine

Le jeu Snake a connu une évolution significative depuis ses débuts. Le concept de base a été développé par le jeu d'arcade Blockade en 1976, qui a été le précurseur des jeux de type "serpent". Le jeu a été intégré dans les téléphones Nokia dans les années 90, notamment le Nokia 6110 et le Nokia 3310, ce qui a permis à Snake d'atteindre un large public.

Principe du jeu :

  • Le joueur contrôle un serpent qui se déplace sur une grille.
  • Le but est de manger des pommes pour grandir et marquer des points.
  • Le jeu se termine si le serpent entre en collision avec les murs ou avec lui-même.

Fonctionnement :

  • Le serpent se déplace automatiquement.
  • Le joueur change la direction (haut / bas / gauche / droite).
  • La nourriture apparaît à une position aléatoire.
  • Chaque nourriture mangée :
    1. augmente la taille du serpent
    2. augmente le score

Représentation des données

  • Serpent : liste de positions (x, y), la tête est le premier élément.
  • Direction : couple (dx, dy) :
    • NORD = (0, -1)
    • SUD = (0, 1)
    • EST = (1, 0)
    • OUEST = (-1, 0)
  • Score : nombre entier.
  • Grille : canvas de 20x20 cases.

Déplacement du serpent

Le déplacement du serpent se fait en ajoutant une nouvelle tête dans la direction choisie. Si le serpent n'a pas mangé de pomme, le dernier carré du serpent est supprimé afin de ne pas avoir un serpent qui grandit infiniment.

Gestion des collisions

  • Collision avec les murs : la tête sort de la grille.
  • Collision avec le corps : la tête touche une autre partie du serpent.
